
# Coffee R Us Admin Portal

## Overview

Coffee R Us Admin Portal is a React Single Page Application that allows administrators to:

- View products
- Search products
- Add products
- Update product prices

## Technologies

- React
- React Router
- JSON Server
- Vitest
- React Testing Library
- Vite

## Installation

```bash
npm install


## Features

- React Router Navigation
- Dynamic Product Search
- Add Products
- Edit Product Pricing
- JSON Server Backend
- Custom Hook
- Context API

## Installation

npm install

## Start Frontend

npm run dev

## Start Backend

npm run server

## Run Tests

npm test
